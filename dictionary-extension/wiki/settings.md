# Settings

## Trigger Settings

- `Show popup on text selection`
- `Selection modifier`
- `Show popup on double-click`
- `Double-click modifier`
- `Enable shortcut trigger`
- `Enable context menu trigger`

Shortcut trigger currently uses `Alt+G`.

## Popup Settings

- `Popup width`
- `Popup height`
- `Default tab`

The browser toolbar popup also uses these dimensions and the same default source-tab preference.

## Translation Settings

- `Translate target language`

Use standard language codes such as:

- `en`
- `vi`
- `ja`
- `ko`
- `zh-CN`

Some common aliases are normalized automatically:

- `vn` -> `vi`
- `jp` -> `ja`
- `kr` -> `ko`
- `cn` -> `zh-CN`
- `tw` -> `zh-TW`

## AI Settings

- `AI base URL`
- `AI API key`
- `AI model`
- `AI prompt template`
- `Preload AI results in background`

The default AI prompt is designed for Vietnamese learners of English and follows the style in `reference.mnd`.

## Supported Prompt Variables

- `{{str}}`
- `{{text}}`
- `{{word_count}}`
- `{{targetLang}}`

No conditional template blocks are supported.
