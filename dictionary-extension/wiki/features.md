# Features

## Implemented

- Translate selected text with Google Translate
- Look up dictionary-style definitions
- Use an AI provider with a custom model, key, base URL, and prompt
- Trigger popup from selection
- Trigger popup from double-click
- Trigger popup from `Alt+G`
- Trigger popup from context menu
- Configure modifiers for selection and double-click triggers
- Configure popup width and height

## Partially Implemented

- Dictionary support
  The current version uses a fallback dictionary source, not an official Google Dictionary backend.

- AI markdown rendering
  The popup supports lightweight formatting, not full markdown.

## Not Implemented Yet

- per-site blacklist and whitelist
- manual input box for custom lookup
- drag-to-resize popup
- configurable keyboard shortcut
- pronunciation audio
- in-popup language switcher
- profiles
- PDF support
